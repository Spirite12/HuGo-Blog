概述

LuaFormatter 是一款 Lua 脚本的代码格式化脚本工具，支持命令行执行，支持可配置项；

执行效果可参考本文中 Git 提交触发代码格式化 目录下的最后一张截图；

支持的IDE有：VsCode，Sublime，Vim；

根据可扩展性，也可应用在Unity、Git钩子内；

当然网络上也有众多 Lua 格式化脚本插件，如：EmmyLuaCodeStyle；

环境

  • Windows 10
  • koihik.vscode-lua-format-1.3.8
  • Unity 2019.4.1f1 (64-bit)
  • Visual Studio Community 2019
  • JetBrains Rider 2019.3.4

获取EXE文件

在作者的GitHub的仓库中并未找到编译好的可执行程序，但是在作者发布的VsCode插件中集成了编译好的版本；

获取步骤如下：

  1. 在 VsCode 插件中搜索 vscode-lua-format 并安装插件；
  2. Windows环境查找路径：%USERPROFILE%\.vscode\extensions\koihik.vscode-lua-format-1.3.8\bin
  3. 便可在以上的路径下的 Win32 文件夹下找到 lua-format.exe 文件

命令行执行

  1. 输入：lua-format -h 可获取到命令行的帮助界面，也可得知是否成功安装LuaFormatter
  2. 命令行输入： lua-format xxx/xxx.lua -c yyy/Lua_Format_Config
    • xxx/xxx.lua 为 Lua 脚本的路径
    • yyy/Lua_Format_Config 为配置脚本的路径，Lua_Format_Config 可自定义命名
    • 注：以上2个路径，可写绝对路径，也可写CMD目录后的相对路径；
  3. 按照指令输入命令行后，会输出格式化好后的文件内容，可根据这个内容，我们便可覆盖原文件做到格式化的处理；

Unity编辑器扩展

思路：根据命令行输出内容，我们可对原文件做写入操作达到Lua脚本格式化的功能；

功能：根据使用者所右击的文件不同则作出不同的处理方式：

  1. 右击Lua文件：则格式化当前脚本
  2. 右击文件夹：则格式化当前文件夹下所有的Lua脚本
  3. 右击非1,2点的文件，则置灰选项，不让点击

using System;
using System.IO;
using GameEditor;
using UnityEditor;
using UnityEngine;

namespace Assets.LuaFormat {
    public class LuaFormat {

        [MenuItem("Assets/Lua LuaFormat", true, 10)]
        public static bool CreateCtrlWinValidate() {
            var theSelection = Selection.activeObject;
            var currentPath = AssetDatabase.GetAssetPath(theSelection);
            bool isLuaEx = Path.GetExtension(currentPath) == LuaExtension;
            bool isFolder = !Path.HasExtension(currentPath);
            return isLuaEx || isFolder;
        }

        [MenuItem("Assets/Lua LuaFormat", false, 10)]
        public static void CreateCtrlWin() {
            var theSelection = Selection.activeObject;
            var currentPath = AssetDatabase.GetAssetPath(theSelection);
            bool isFolder = !Path.HasExtension(currentPath);

            if (!isFolder) {
                FileRunCmdLine(Environment.CurrentDirectory + "\\" + currentPath);
            }else {
                //对文件夹下所有Lua文件做格式化处理
                DirectoryInfo direction = new DirectoryInfo(currentPath);
                FileInfo[] files = direction.GetFiles("*", SearchOption.AllDirectories);
                foreach (var file in files) {
                    if (file.Name.EndsWith(LuaExtension)) {
                        FileRunCmdLine(file.FullName);
                    }
                }
            }
        }

        /// <summary>
        /// 根据路径执行CMD命令
        /// </summary>
        private static void FileRunCmdLine(string filePath) {
            string configPath = Environment.CurrentDirectory + ConfigPath;
            string cmd = filePath + " -c " + configPath;
            // 执行 cmd
            string fileOutput = EditorUtil.RunCmdLine(LuaFormatter, cmd);
            // todo ：输出文件格式化内容后，可进行读写当前文件
            Debug.LogWarning(fileOutput);
        }

        private const string LuaFormatter = "lua-format";
        private const string ConfigPath = "\\Assets\\LuaFormat\\LuaFormatter\\Lua_Format_Config";
        private const string LuaExtension = ".lua";
    }
}

Git 提交触发代码格式化

思路：在提交信息的时候，利用 Git 钩子 pre-commit 触发代码对在暂存区的文件内，查找 Lua 文件并对其进行 LuaFormatter 代码格式化，并成功触发提交；

步骤：

  1. 进入当前工程根目录的 .git\hooks 文件夹下寻找 pre-commit.sample 文件，复制新一份，并对新文件命名为 pre-commit，即可在提交 Git 的时候触发该文件代码；

  2. 对 pre-commit文件内添加如下代码：

    #!/bin/bash
echo "pre-commit lua formatter"

#Lua文件的后缀扩展名
lua_extend="lua"
#Lua_Format_Config文件的绝对路径
Lua_Format_Config=""

result=$?
if [ $result != 0 ]; then
    echo "=============================="
    echo "lua formatter fail, result is : "$result
fi

cur_dir=$(pwd)
cached_list=$(git diff --cached --name-only)
for file in ${cached_list[@]}
        do
            pattern=$(expr "$file" : '.*\.\(.*\)')
            if [[ $lua_extend == $pattern ]]; then
                #执行LuaFormatter
                Lua_Format_Path=$cur_dir"/"$file
                Lua_Format_Path_Temp=$cur_dir"/"$file"_temp"
                lua-format $Lua_Format_Path -c $Lua_Format_Config > $Lua_Format_Path_Temp
                lua-format $Lua_Format_Path_Temp -c $Lua_Format_Config > $Lua_Format_Path
                rm $Lua_Format_Path_Temp

                #修改添加暂存区
                git add $file
            fi
        done

exit

$result
