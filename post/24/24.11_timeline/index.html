前言

  • 三种动画播放的理解：
    • Animation：此组件可以让挂载的物体，进行动画播放；适合物体只需要播放单个动画；
    • Animator：则是拥有多个 Animation 组件，并可以设计在某条件下播放指定 Animation；适合角色的动画状态切换；
    • Timeline：控制多个物体，并按照时间线依次播放所有物体的动画；适合用来剧情播放；
    • 总结：从 Animation -> Animator -> Timeline ，使得动画的操作范围更广，更加方便了开发者的使用；当然的在不同的场合下，选择适合的动画组件才是更优的解决思路；
  • 如果你有用过 “会声会影” 此类的视频剪辑软件，你就会发现此类软件的编辑界面与操作习惯都与Timeline 的使用大同小异；

环境安装

  • Unity 版本： 2021.3.6f1c1

  • 安装：

    • 路径：菜单栏 -> Window -> Package Manager

    • 窗口：右上角搜索 Timeline -> 右下角点击安装 Install

  • 导入实例：

    • 安装完成后，在此页面导入相关实例查看教程

基础说明

创建 Timeline 资源

  1. 创建资源：
    • 在 Project 区域点击右键 -> Create -> Timeline ；
    • 将创建好的实力命名为 Timeline；
  2. 查看属性：点击当前 Timeline，在 Inspector 查看属性；
    • Frame Rate ：帧速率，此参数的修改对播放速度、关键帧、轨道或剪辑没有影响；
    • Duration Mode：持续时间模式
      • Based On Clips：根据 Timeline 内最后一帧的时间设置为当前资源的最长时间；
      • Fixed Length：手动填写想要的时长；
    • Duration： 1）s ：时间（Second），以秒为单位 ；2）f：帧率（Frame）= Frame Rate 填写的帧数 * s
    • Scene Preview：场景预览，建议勾选；

添加 Timeline 组件

  1. 在 Hierarchy 面板上创建物体，并命名为 Timeline，添加组件 Playable Director；
  2. 查看组件，并在参数 Playable 赋值刚才创建好的 Timeline 资源；
  3. 参数说明：链接

打开 Timeline 窗口

  1. ①：播放控件，链接

  2. ②：曲目列表：

    • 添加轨道：XXX Track

      • Activation Track：激活轨道，在绿色所划分的时间区域内，当前物体的激活状态为设置为 Ture；
      • Animation Track：动画轨道，面板上需要赋值需要控制的Animator，右边则是按照时间线赋值 Animation 给 Animator 并进行播放；
        • Staff_vs_Staff_OneFight_MaleA 是示例中的预制件；
      • Audio Track：音频轨道，在黄色区域内会播放 AudioClip ，需要在面板上指定 Audio Source 组件，以便接受音频；
        • hey 是示例中的音频；
      • Control Track：控制轨道；
        • Video Track ：把 M09-1317 视频拖入窗口内，并可按照时间线播放；
        • 也可在 Hierarchy 面板内拖入 DustMotesEffect ，来达到粒子的播放；
          • DustMotesEffect 是示例中的预制件
        • 当然可控制的不止这样；
      • Signal Track：信号轨道，在右侧时间线内接收到信号后，则会去执行信号轨道组件内的操作；当前操作则是 SetActive 组件的显隐操作；左边赋值需要赋值带有 Signal Receive 组件的物体；
      • Playable Track：播放组件，使用此轨道添加可播放剪辑。每个可播放剪辑都绑定到一个脚本，该脚本使用可播放 API创建自定义动画工具、效果或游戏机制。请注意，脚本必须继承自PlayableAsset才能自动检测并创建可播放轨道。
      • 当然你也可以再嵌套一个子 Timeline 进行播放；
        • 示例可以参考名为：GameplaySequence 的Timeline；

    • 添加轨道组：Track Group

      • 轨道组可以将多个 Track 归纳到一个组下面，作用是：
        1. 轨道分组、分类便于开发查看视图；
        2. 统一管理当前轨道组下的 Track 的上锁、是否播放；

  3. ③：编辑模式， 链接

  4. ④：是否显示标注；

    • 类似代码说明一样的标识；点击则可以查看注释，取消则无法选中；

    • 标注右边的2个图标说明：链接

      • 钥匙图标：是否需要上锁，上锁则无法对当前轨道进行操作；
      • 眼睛图标：是否需要显示，勾选为不显示，则播放Timeline的时候会不播放当前轨道的内容；

  5. ⑤：时间线，链接

    • 显示当前时间线内播放的物体是啥；

示例介绍

Timeline_CustomTracks

  1. 打开示例中命名为：Timeline_CustomTracks 的场景；

  2. 点击 Hierarchy 面板上名为 Timeline 的物体，双击打开当前物体 Playable Director 组件上的 TimeLine 实例窗口；

  3. 面板内轨道说明：

    • Time Dilation Track ：控制组件，当进入 TimeWarp 的时候，则会进行 TimeScale 的变化；影响游戏的快慢进度；
    • Video Track：控制轨道，按照时间线播放视频；
    • Text（TMP）：播放轨道，按照时间线内更改 Tmp 组件的文本；
    • Tween Track：轨道组
      • Arrow：激活轨道，按照时间线激活物体；
      • Arrow：控制轨道，按照时间线控制当前物体的 Transform 组件位移；
    • Character Tracks：轨道组
      • Staff_vs_Staff_OneFight_MaleA：动画轨道，按照时间线播放动画；
      • Staff_vs_Staff_OneFight_MaleB：同上

  4. 其他说明：

    • Timeline 组件会列出当前资源下的动画资源；

    • Playable Director 组件内也会列出当前绑定的轨道；
