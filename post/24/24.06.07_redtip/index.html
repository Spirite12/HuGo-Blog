使用示例

一、红点初始化

  • RedTipMgr ：初始化函数为： Init 和 Destroy 的函数；

  • RedTipConst：初始化函数为：Initialization，此函数的目的是往 RedTipTree 的相关存储变量内赋值；

    • 因此，当我们在场景中挂载 MainUI 脚本，当游戏运行的时候，需要执行的代码如下：

      using System.Collections;
using Module;
using UnityEngine;
namespace Base {
    public class MainUI : MonoSingleton<MainUI> {

        private void Awake() {
            RedTipConst.Initialization();
            RedTipMain redTip = new RedTipMain(RedTipConst.RedTipMain, null);
            RedTipMgr.Init(redTip);
        }

        void Destroy() {
            RedTipMgr.Destroy();
        }
    }
}

二、新增红点

  • 当新增一个需要红点树控制的红点类的时候，需要执行以下的操作：
    • 若新增红点名为：RedTipMain
    • RedTipMain
      • 新增 RedTipMain.cs 的脚本，脚本使用参考第三点；
    • RedTipConst
      • 添加相对应的红点常量：
        • 例如：public const string RedTipMain = nameof(RedTipMain);
      • 在 redTipTreeDic 字典内添加红点父子关系数据：
        • 例如：{RedTipMain, new List<string>() { RedTipMain1, RedTipMain2 }},
      • 在 redTipBaseDic 字典内添加实例化红点的操作数据；
        • 例如：{RedTipMain, ()=> new RedTipMain()},
  • 当新增一个无法用红点树控制的红点类的时候，需要手动写调用脚本；
    • 距离可以参考第五点；

    • 在 Destroy 函数内：可以做事件移除、数据销毁等操作

四、红点UI显示

  • 在任一物体上挂载 RedTipUIBind 脚本，并对其填写 RedTipName 与 RedTipId 或者进行 InitRedTipNameAndId 的函数赋值便可；

  • 即可实现：红点的UI显示与数据更新；

五、红点ID使用

  • 使用场景：在一系列的系统玩法中，红点的逻辑都一致，只是相对应的ID不同，则可以由此来区分并处理；

  • 注意事项：由于这类红点没办法在红点树内就初始化好，因此需要开发人员手动调用；

  • 以下为创建的实例代码：

    • 找到根节点：RedTipMain
    • 为根节点添加2个名为 RedTipMain1 但 Id 不同的 1,2 红点；
    • 实际逻辑的话便在 RedTipMain1 类中按照参考第三点来完成红点类的代码编写；

    public class MainUI : MonoSingleton<MainUI> {

    private void Awake() {
        RedTipConst.Initialization();
        RedTipMain redTip = new RedTipMain(RedTipConst.RedTipMain, null);
        RedTipMgr.Init(redTip);
    }

    private void Start() {
        var redTipMain = RedTipMgr.GetRedTip(RedTipConst.RedTipMain);

        var redTipA = RedTipTree.GetRedTipClass(RedTipConst.RedTipMain1);
        var redTipAId = 1;
        redTipA.SetNameAndParent(RedTipConst.RedTipMain21, redTipMain, redTipAId);
        redTipMain.CreateChildRedTip(RedTipConst.RedTipMain21, redTipA, redTipAId);
        redTipA.CreateAllChildRedTip();

        var redTipB = RedTipTree.GetRedTipClass(RedTipConst.RedTipMain1);
        var redTipBId = 2;
        redTipB.SetNameAndParent(RedTipConst.RedTipMain21, redTipMain, redTipBId);
        redTipMain.CreateChildRedTip(RedTipConst.RedTipMain21, redTipB, redTipBId);
        redTipB.CreateAllChildRedTip();
    }
}
