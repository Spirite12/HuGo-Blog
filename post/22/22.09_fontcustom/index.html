Font 自定义字体

2022-09-12
Unity

前言

  • 在游戏开发阶段，项目中难免会用到一些自定义的字体。这些字体大部分会用在数字，字母以及相关字符的显示上，而显示的样式则由美术决定的，对此则使用到了Unity的Font知识；

  • 以下为实现效果的截图和fontsettings资源的部分参数截图：

素材准备

  • 从素材网下载美术贴图资源：资源地址
    • 如有侵权，请发邮件告知；
  • 导入Unity后，创建Material；
  • Material的Shader选择GUI Shader 并将下载的贴图赋值到Texture中；
  • 右键工程选择 Create -> Custom Font，将新创建的 Material 赋值到Default Material；
  • 设置相关参数后，便可在Text组件中选择Font为新创建的字体，并输入相对应的字符便可实现效果；

参数介绍

  • Line Spacing ：行间距

  • Default Material ：默认材质球

  • Ascii Start Offset ：初始字符的偏移量，默认为0便可

    • 默认写0，原本48的Ascii则对应数字0；若写1，则49的Ascill则对应数字0；

  • Track：同行字符之间的接近程度比例；

  • Character Rects

    • Size ：个数

      • Index：Ascii 表上的对应的索引值

      • Uv

        • X、Y：以贴图左下角为（0,0）点，右上角为（1,1）点，当前字符的左下角为目标点。算出宽度1，高度1的当前字符所占的比例位置；
        • W、H：宽度、高度。即为1除以一行/列所展示的字符数；

      • Vert

        • X、Y：当前字符的X、Y偏移像素值；
        • W、H：当前字符的所占的宽度和高度像素值；

      • Advance ：当前字符的宽度像素值；（在计算实际距离时，应将其乘以 Track）

      • Flipped：是否旋转90度；
